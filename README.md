# projeto-academia
> Projeto Academia é uma aplicação front-end que apresenta uma landing page para academia, além de páginas de login e cadastro com validação básica em JavaScript.

## Tecnologias Utilizadas
  * HTML
  * CSS
  * JavaScript

## Funcionalidades
* Landing page com navegação
* Página de cadastro com validação de campos
* Página de login com verificação básica de formulário

## Como Utilizar
```
git clone https://github.com/matheusnsilvab/landing-page-academia
```
```
cd landing-page-academia
```
